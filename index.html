About This Site

This website was made by Noah Eisen and Robert Rose. Our initial goal was to improve the performance and aesthetics of this site, which we both used while reading Ulysses. Rob made the initial version with the Flask library. His code can be found here. Noah then rewrote all of the backend in Javascript, so this site can run in browser. Noah also extended the project to include more publicly available works. His code can be found here. Hope you enjoy the site!
